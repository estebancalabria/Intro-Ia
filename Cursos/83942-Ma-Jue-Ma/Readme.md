# Concepto 

* System Prompt :es el mensaje inicial que se le da a un modelo de inteligencia artificial para establecer el contexto, las reglas y el comportamiento con el que debe responder. Funciona como una “personalidad base” o un conjunto de instrucciones que el modelo seguirá durante toda la conversación, guiando su tono, sus límites, sus objetivos y la forma en que procesa las preguntas del usuario.
