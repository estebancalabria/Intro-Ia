# Glosario

* Fremium
  * Es gratis pero con ciertos limites. Superas ese limite y tenes que pagar para seguir usando.
  * Es gratuita pero tenes abonos/planes para mejoras como puede ser el mayor tiempo
* Un system prompt son las instrucciones personalizadas que cada empresa le da a un LLM y define su "peronalidad" y como responde.
 * Agente
     * Angentes Conversacionales
         * Un chatbot (como chatgpt) pero especializad en un a tarea, con conocimiensto especifico y cierta capacidad de accion mas alla de un chatbot convencional
         * Ej: Servicio al cliente, Soporte tecnico especifico de una empresa, Abogado con conocimiento de la ley argentina
         * Chatbot como ChatGPT + Instrucciones personalizadas + Conocimientos Especifico de una area de conocimiento + herramientas que le permiten hacer cosas
         * Se puede hacer un agente por ejemplo con los Gemmas de Gemini
     * Agentes Autonomos
