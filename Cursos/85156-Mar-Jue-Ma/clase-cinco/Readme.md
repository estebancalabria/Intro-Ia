# Glosario

* Human in The loop : Todo proceso que incluye la IA como herramienta de productividad incluya una persona validando todo el proceso
* No Deterministica : Mismo prompt -> Salidas Distintas
* Grounding : El concepto de verificar o relacionar la respuesta de la IA con datos reales
