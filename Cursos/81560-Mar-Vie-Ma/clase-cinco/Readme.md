## Generacion de Imagenes

- ### Texto a Imagen

* Se pueden generar imagenes desde Texto con ChatGPT, Gemini (Nano Bannana)
* Podemos implementar este Caso de Uso con LeonardoAI
  * https://leonardo.ai/
  * Es practicamente gratuito : 150 creditos diarios
* **TIP: Los modelos de Generacion de Imagenes funcionan mucho mejor con prompts en Ingles**
*  Para el prompt se lo pido a ChatGPT

```
Futuristic transforming robot, highly detailed mechanical design, metallic textures, glowing energy cores, dynamic action pose, cinematic lighting, epic sci-fi background, hyper-realistic style, ultra-detailed, 8K resolution, dramatic perspective
```

* Puntaje : 10 / 10
