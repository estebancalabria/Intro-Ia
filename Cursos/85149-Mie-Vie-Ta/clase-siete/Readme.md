 #### Generacion de Imagenes a partir de texto / Imagen

* URL
  * https://leonardo.ai/
* Caracteristicas
  * Incluye los modelos de generacion de imagenes mas populares
  * Recarga los creditos diariamente
  * Tiene muchas mas cosas que generacion de imagenes

* Tips
    * Para mejores resultados conviene siempre los prompts en ingles

```
High-quality cinematic image of a clothing printing company where t-shirts are being screen printed, and in the corner, a happy model is trying on a t-shirt and absolutely loving it.
```
