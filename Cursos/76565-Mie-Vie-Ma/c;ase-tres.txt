# Prompt Engineering
   - System Prompt
         -Alimentando  ChatGPT con un system prompt especifico puedo generar mis propios GPTs
   - Memoria   
   - Plantilla de prompt : Un prompt reutilizable que tiene algunos fragmentos generalmente entre corchetes [] a completer antes de utilizarlos

   - Contexto
       - TIP : explayarse en los prompts (Utilizar la voz)
       - Utilizar las conversaciones en forma inteligente para manejar major el context
       - Alimentar a la IA con el contexto necesario antes de preguntarle lo que queremos
       - Prompt Chainning : Encadenar los prompts para armar una conversacion coherente con el major contexto para mi objetivo

   - Patrones de Prompting
       - Persona
            - Tip : Utilizar ChatGPT como expert en Prompt Engineering
