## Glosario

* Prompt : La instruccion que uno le da al modelo de lenguaje
* System Prompt : Parte del cerebro invisible que usa chat GPT. El system prompt es la instrucción base que define cómo debe comportarse el modelo durante una conversación.
* Prompt Engineerings : Rama que estudia como hablarle a la ia y escribir prompts efectivos para obtener mejores respuestas del modelo de lenguaje
* Alucinar : Cuando la IA se ve de tema y me responde cualqueir cosa. Cuando se va por las Ramas.
* Contexto : Seria toda la conversacion
