# 🧪 LABORATORIO — Evaluando Modelos Propietarios de IA

## 🎯 Introducción

Imaginá que sos parte del **equipo de innovación de una empresa** que está considerando implementar un **Large Language Model (LLM)** para distintos procesos corporativos (atención al cliente, investigación, generación de reportes, productividad, desarrollo de software o contenido creativo).

El objetivo de este laboratorio es:

* Explorar cada modelo propietario uno por uno.
* Descubrir en qué destaca cada modelo según sus fortalezas y limitaciones.
* Analizar cómo cada modelo puede aportar valor a la empresa según su enfoque y capacidades.
* Preparar una recomendación fundamentada sobre qué modelo conviene implementar en función de casos de uso reales.

---

## 1️⃣ ChatGPT (OpenAI)

**Fortaleza principal:** Lenguaje natural.
**Ideal para empresas que prioricen:** Modelos que se destaquen en comunicación basada en lenguaje humano.

**Pasos para explorar:**

1. Abrí ChatGPT en [https://chatgpt.com/](https://chatgpt.com/)
2. Probá preguntas abiertas y solicitá distintos roles o estilos.
3. **NOTA:** Observa cómo este modelo destaca en **lenguaje natural**.
