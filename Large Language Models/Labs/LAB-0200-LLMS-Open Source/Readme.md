# 🧪 LABORATORIO 2 — Evaluando Modelos Open Source de IA

## 🎯 Introducción

Imaginá que sos parte del **equipo de innovación de una empresa** que **valora la privacidad y la seguridad de sus datos**. La empresa quiere evaluar distintos **modelos de lenguaje open source** que se pueden **instalar localmente en sus propios servidores**, evitando enviar información sensible a terceros.

El objetivo de este laboratorio es:

* Explorar distintos modelos open source de manera progresiva.
* Descubrir **en qué destaca cada modelo** según sus capacidades técnicas y enfoque.
* Evaluar **fortalezas y limitaciones** en términos de **privacidad, control y seguridad**.
* Practicar **uso de prompts complejos y personalizados**, incluyendo razonamiento lógico, memoria y roles.
* Preparar un **resumen comparativo** sobre qué modelo conviene para la empresa según **seguridad, privacidad y capacidades técnicas**.

---

## 1️⃣ Llama (Meta/Facebook)

**Fortaleza principal:** Flexibilidad de despliegue local y control sobre el modelo.
**Ideal para empresas que prioricen:** Modelos que se puedan instalar y controlar internamente.

**Pasos para explorar:**

1. Accedé al modelo Llama a través de su implementación en **Hugging Face**.
2. Probá generación de texto y observá comportamiento con distintos prompts.
3. **NOTA:** Observa cómo este modelo destaca en **flexibilidad para despliegue local y control interno**.

---

## 2️⃣ Gemma (Google DeepMind)

**Fortaleza principal:** Familia de modelos open source, livianos y diseñados para integraciones.
**Ideal para empresas que prioricen:** Uso por desarrolladores, investigadores o integración dentro de apps propias.

**Pasos para explorar:**

1. Accedé a los modelos Gemma desde **Hugging Face**.
2. Probá generación de texto y verificá consistencia y velocidad de respuesta.
3. **NOTA:** Observa cómo este modelo destaca en **liviano, open source y apto para integración en apps**.

---

## 3️⃣ DeepSeek (China)

**Fortaleza principal:** Introducción del “Modo Pensamiento” para razonamiento lógico.
**Ideal para empresas que prioricen:** Resolución de problemas matemáticos o de lógica usando **Cadena de Pensamiento** (Chain-of-Thought).

**Pasos para explorar:**

1. Abrí [DeepSeek](https://www.deepseek.com/en/) o [chat.deepseek.com](https://chat.deepseek.com/)
2. Probá problemas de razonamiento lógico o matemático. Ejemplo:
   *“Hay tres personas en una habitación. Uno de ellos es un asesino. Entra una cuarta persona y mata al asesino. ¿Cuántos asesinos vivos quedan ahora?”*
3. **NOTA:** Observa cómo este modelo destaca en **razonamiento lógico mediante Cadena de Pensamiento**.

---

## 4️⃣ Qwen (Alibaba)

**Fortaleza principal:** Copia open source de ChatGPT con memoria y personalización de instrucciones.
**Ideal para empresas que prioricen:** Uso extensivo sin límites, personalización y control interno del modelo.

**Pasos para explorar:**

1. Abrí [Qwen](https://chat.qwen.ai/)
2. Probá prompts personalizados y verificá cómo maneja la memoria.
3. **NOTA:** Observa cómo este modelo destaca en **memoria, personalización y disponibilidad ilimitada**.

---

## 💡 Extra — Ejecución local y Hugging Face

Todos estos modelos **open source** se pueden subir a **Hugging Face** y ejecutarlos localmente usando herramientas como **LMStudio**, lo que permite:

* Mantener **control total sobre los datos**.
* Experimentar sin depender de la nube o límites de uso.
* Probar **personalización avanzada y tareas específicas** en un entorno seguro.
