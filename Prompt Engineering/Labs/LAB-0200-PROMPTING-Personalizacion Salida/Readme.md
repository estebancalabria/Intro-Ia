# 🧪 LABORATORIO

# **🚀 Transformá Datos de Productos Tecnológicos con IA**

## 🎯 Objetivo

Vas a simular un caso real de empresa: generar información sobre los **10 productos tecnológicos más vendidos en MercadoLibre** y transformarla para distintos equipos, usando **formatos profesionales**.

Durante este laboratorio vas a ver cómo **una misma información** puede viajar por distintos formatos: lista, JSON, XML, CSV, Markdown (Word), HTML (PDF) y visualización Mermaid. Todo con IA.

💡 Esto es lo que hacen en empresas tecnológicas y consultoras para que los datos lleguen **listos para cada área**.

---

# 🧩 Estructura del Laboratorio

---

## 0️⃣ Investigación Inicial — Conociendo las herramientas 🔍

Antes de empezar, explorá:

* Qué modelos de IA permiten **generar y transformar datos**: ChatGPT, Claude, Groq.
* Qué formatos se usan profesionalmente: JSON, XML, CSV, Markdown, HTML, PDF, Mermaid.
* Cómo estos formatos se integran con sistemas, documentación y reportes corporativos.

✨ **Tip:** observar el flujo de datos en una empresa te ayuda a entender **para qué sirve cada formato**.

**Aquí aprenderás a:**

* Reconocer qué formato conviene para cada área.
* Entender por qué elegir bien el formato es **parte del trabajo profesional**.

---

## 1️⃣ Generación de la Lista Inicial 📝

Prompt base (para copiar y pegar):

```
Dame una lista de los 10 productos tecnológicos más vendidos en MercadoLibre en Latinoamérica que podrían ser buenos candidatos para vender en una empresa de tecnología.

Para cada producto indicar:
- nombre del producto
- categoría
- precio promedio
- público objetivo
- característica principal
```

**Aquí aprenderás a:**

* Pedir información clara y completa a la IA.
* Crear un **dataset base** que luego se puede transformar.
* Ver cómo un LLM genera información **lista para un caso profesional real**.

---

## 2️⃣ JSON — Datos para Desarrollo 💻

**Contexto:** El equipo de desarrollo necesita **cargar los datos en un sistema o API**.

Prompt listo:

```
Convertí la lista de productos a formato JSON con los campos:
nombre, categoria, precio_promedio, publico_objetivo, caracteristica_principal.
Solo devolver el JSON.
```

**Aquí aprenderás a:**

* Transformar la lista en un formato **que las máquinas pueden leer**.
* Ahorrar tiempo en tareas repetitivas.
* Preparar datos para integraciones y APIs.

---

## 3️⃣ XML — Para Sistemas Legacy 🏛️

**Contexto:** Algunos sistemas antiguos solo aceptan XML.

Prompt listo:

```
Convertí la lista de productos a formato XML manteniendo la misma estructura de datos.
Solo devolver el XML.
```

**Aquí aprenderás a:**

* Adaptar información a sistemas más antiguos.
* Mantener jerarquía y organización de los datos.
* Ser flexible con los formatos que exige cada área de la empresa.

---

## 4️⃣ CSV — Para Analistas 📊

**Contexto:** El equipo de analistas quiere usar los datos en **Excel o Google Sheets**.

Prompt listo:

```
Convertí la lista de productos a CSV con columnas:
nombre, categoria, precio_promedio, publico_objetivo, caracteristica_principal
```

Luego:

1. Copiá el CSV
2. Pegalo en Excel o Google Sheets

**Aquí aprenderás a:**

* Preparar datos listos para **gráficos, dashboards y análisis**.
* Ver cómo la IA te ayuda a **ahorrar tiempo**.
* Transformar información para un trabajo práctico y visual.

---

## 5️⃣ Markdown — Para Documentación / Word 📄

**Contexto:** El equipo de marketing necesita un **documento profesional**. Markdown es un lenguaje de marcado ligero que permite crear documentos claros y fáciles de convertir a Word, PDF o HTML.

Más info: [https://es.wikipedia.org/wiki/Markdown](https://es.wikipedia.org/wiki/Markdown)

Prompt listo:

```
Generá un documento en Markdown para cada producto usando la plantilla:

# [PRODUCTO]

## Características
* Categoría: 
* Precio promedio: 
* Público objetivo: 
* Característica principal:

## Descripción
Breve explicación del producto y por qué es popular.

## Oportunidad de negocio
Por qué sería bueno venderlo en la empresa.
```

### Integración con Word

1. Copiá el Markdown
2. Pegalo en Microsoft Word

💡 **Tip:** Word mantiene títulos, listas y jerarquía automáticamente. ¡Listo para entregar!

**Aquí aprenderás a:**

* Crear documentación profesional sin esfuerzo.
* Mantener jerarquía y estructura clara.
* Transformar información en reportes listos para cualquier área.

---

## 6️⃣ HTML — Para Reportes / PDF 🖥️

**Contexto:** La dirección quiere un **reporte elegante**. HTML permite crear documentos bonitos que se pueden exportar a PDF.

Prompt listo:

```
Generá un documento HTML moderno que muestre la lista de productos y sus características.
Incluir título, tarjetas para cada producto, tipografía moderna y estilo limpio y profesional.
```

Luego:

1. Guardar HTML
2. Abrir en navegador
3. Imprimir → Guardar como PDF

**Aquí aprenderás a:**

* Crear reportes visualmente atractivos.
* Preparar entregables que se vean profesionales para clientes o superiores.
* Usar HTML como herramienta de presentación.

---

## 7️⃣ Diagrama Mermaid — Visualización Ejecutiva 📈

**Contexto:** La dirección quiere ver **de un vistazo la distribución de productos por categoría**.

Prompt listo:

```
Generá un diagrama Mermaid de pie mostrando la distribución de productos por categoría.
```

Luego:

1. Copiá el diagrama
2. Abrí [https://mermaid.live/](https://mermaid.live/)
3. Visualizalo

**Aquí aprenderás a:**

* Transformar datos en visualizaciones claras y rápidas.
* Comunicar información compleja de manera simple.
* Usar IA para **ahorrar tiempo y generar gráficos profesionales**.

---

## 🎯 Resultado Esperado

Al finalizar, deberías tener:

1. Lista inicial de productos
2. JSON para sistemas
3. XML para integraciones
4. CSV para Excel / Sheets
5. Markdown integrado en Word
6. HTML exportable a PDF
7. Diagrama Mermaid de la distribución por categoría

Esto te muestra **cómo un solo set de información puede adaptarse a todos los equipos de una empresa usando IA**.
